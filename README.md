# mocopiVR

mocopiとMetaQuestProを活用したオンライン型脱出アトラクションです。

開発環境：
Unity ver2021.2.11
Photon
mocopiSDK
Oculus integration

操作方法：
・もも上げすると視点を向いてる方向に移動
・トリガーボタンで掴む

パックヤード側：
・スペースキーで準備完了（2人プレイの時）
・Cキーで一人でも強制スタート
